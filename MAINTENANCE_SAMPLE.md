## Testing Instructions:

1. Navigate to the System Admin panel
2. Go to the System Configuration section
3. Find the "Maintenance Notice" form
4. Check the "Enable Maintenance Notice" checkbox
5. Fill in the fields with the sample data above
6. Click "Save Maintenance Notice"
7. Verify the notice appears in the customer portal

## Notes:

- Date format should be: `YYYY-MM-DDTHH:mm` (e.g., `2024-12-20T08:00`)
- Make sure the end date/time is after the start date/time
- The notice will only be visible to customers if "Enable Maintenance Notice" is checked
- You can test by viewing the customer portal after saving
